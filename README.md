# 字媒体卡片应用

一个将文字转换为精美卡片的在线工具，支持多种模板、自定义样式和导出分享。

## 功能特点

- 输入文字，一键生成精美文字卡片
- 多种精美预设模板
- 自定义字体、颜色、大小等文字样式
- 支持纯色背景和渐变背景
- 添加贴纸元素装饰卡片
- 自定义卡片尺寸
- 导出图片，方便分享到社交媒体

## 技术栈

- Vue 3 + TypeScript
- Element Plus UI 组件库
- UnoCSS 原子化 CSS 框架
- Pinia 状态管理
- html2canvas 用于图片导出

## 开发指南

### 安装依赖

```bash
npm install
```

### 本地开发

```bash
npm run dev
```

### 构建项目

```bash
npm run build
```

## 使用方法

1. 在首页输入需要展示的文字内容（限制在100字以内）
2. 点击"生成卡片"进入编辑页面
3. 在左侧选择喜欢的模板
4. 在右侧面板调整文字样式、背景和添加贴纸
5. 调整合适的卡片尺寸
6. 点击"导出图片"保存成PNG格式
